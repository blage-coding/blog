# [26. 删除有序数组中的重复项](https://leetcode.cn/problems/remove-duplicates-from-sorted-array/)

解题思路：双指针

```java
class Solution {
    public int removeDuplicates(int[] nums) {
        int slow=0;
        int fast=0;
        while(fast<nums.length){
            while(fast+1<nums.length&&nums[fast]==nums[fast+1]) fast++;
            nums[slow]=nums[fast];
            slow++;
            fast++;
        }
        return slow;
    }
}
```
