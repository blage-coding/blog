# 64.最小路径和

给定一个包含非负整数的 `*m* x *n*` 网格 `grid` ，请找出一条从左上角到右下角的路径，使得路径上的数字总和为最小。**说明：**每次只能向下或者向右移动一步。

> 输入：grid = [[1,2,3],[4,5,6]]
> 输出：12

1. 动规。dp[i\][j\]表示从起点到grid[i\][j\]位置,初始化第零行第零列，**dp[i\][j\]=Math.min(dp[i\][j-1\]+dp[i-1\][j\])+grid[i\][j\]**。
